Kami menyambut semua jenis kontribusi, tanpa memandang besar kecilnya atau tingkat keahlian Anda. Sebagai proyek open source, kami percaya bahwa kontribusi layak mendapatkan apresiasi. Kami dengan senang hati membantu Anda dalam membuat pull request (PR), menulis dokumentasi, dan mewujudkan ide fitur apa pun.

Ingin terlibat lebih jauh? Lihat [dokumen Tata Kelola](https://github.com/withastro/.github/blob/main/GOVERNANCE.md) untuk detail peran yang tersedia, proses nominasi pengelola, proses review kode, dan penerapan kode etik.

## Cara Berkontribusi

### Repositori Proyek

Ada banyak cara untuk berkontribusi pada proyek Astro! Setiap repositori Astro menyertakan tautan ke file `CONTRIBUTING.md` di bagian root proyek melalui README-nya.

Kunjungi [profil GitHub Astro](https://github.com/withastro) dan jelajahi repositori berikut:

- [Kode dasar Astro](https://github.com/withastro/astro), termasuk integrasi resmi dan template awal

- [Dokumentasi Astro](https://github.com/withastro/docs), yaitu seluruh situs dokumentasi. Anda bisa berkontribusi pada konten tertulis, atau juga pada aspek aksesibilitas, CSS, UI, dan UX. Karena dokumentasi tersedia dalam berbagai bahasa, bantuan untuk menerjemahkan juga sangat kami butuhkan.

- [Kompiler Astro](https://github.com/withastro/compiler), ditulis dalam Go dan dibundel ke dalam WASM

- [Language Tools Astro](https://github.com/withastro/language-tools), alat bantu editor untuk bahasa `.astro`

- [Starlight](https://github.com/withastro/starlight), framework dokumentasi resmi untuk Astro

- [Roadmap Astro](https://github.com/withastro/roadmap), tempat masa depan Astro dirancang: ide, usulan, dan RFC resmi

### Jenis Kontribusi

Selain menyumbangkan kode atau konten, Anda juga bisa membuat dampak besar dengan meninggalkan komentar review pada PR, memberi ide pada isu dan diskusi GitHub yang ada, atau ikut serta dalam tugas perawatan pada isu yang ditandai sebagai “pinned”.

Semua PR, terutama PR terjemahan, memerlukan reviewer! Meninjau PR dan meninggalkan komentar, saran, atau bahkan pesan sederhana seperti “LGTM!” (“Looks Good To Me!”) adalah cara yang bagus untuk memulai dan mengenal lebih dalam tentang Astro.

Kami juga memiliki [Discord](https://astro.build/chat) komunitas yang sangat aktif! Kami menghargai kontribusi berupa menjawab pertanyaan, menyambut anggota baru, serta membagikan proyek atau alat yang dibuat dengan atau untuk Astro. Selain kontribusi teknis, kami juga menghargai partisipasi komunitas dalam bentuk blog, video, presentasi, dan membantu menjaga komunitas tetap sehat.

## Berkontribusi pada Dokumentasi

Kami menyediakan berbagai panduan untuk membantu Anda berkontribusi ke dokumentasi Astro.

Apakah Anda baru pertama kali berkontribusi ke open source, ingin menambahkan dokumentasi untuk fitur baru yang Anda buat, penerjemah berpengalaman yang mencari halaman berikutnya untuk diterjemahkan, atau ingin tahu lebih banyak tentang cara menjadi reviewer PR — kami siap membantu!

Kunjungi situs [*Dokumentasi Kontributor Astro*](https://contribute.docs.astro.build), tempat Anda dapat menemukan panduan lengkap untuk editor, penulis, penerjemah, pengembang fitur, dan reviewer.
